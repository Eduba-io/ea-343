# UPAGO x Eduba, bespoke landing page

Intended URL path: /for/upago

## What this is

A single-page read built for a conversation at booth SS98 (Startup Spotlight, eMerge Americas 2026) with UPAGO, the LATAM fintech specializing in revenue assurance and online collections.

The page uses UPAGO's own brand tokens (navy `#0a2442`, the green-to-blue diagonal gradient, Montserrat, soft blue `#E8F4FF` surfaces, rounded 1rem cards) so it reads as native to their site. It carries the pitch body from section 7 of the fit analysis: the 60/30/10 layer frame, the VigilOre precedent, the Ethics Engine paper, and a clear next step.

## Language

UPAGO is a Spanish-first company (Chile, Mexico, Colombia). The page defaults to Spanish and offers an ES / EN toggle in the top bar. The English version mirrors the structure so a bilingual prospect can flip between them without losing their place.

## Files

- `index.html`. Page markup, inline bilingual copy, ES default.
- `styles.css`. All brand tokens and layout.
- `logo.svg`. UPAGO logo, copied from `../Brand Pack/assets/logo.svg`.

## Hosting

Single static page. No build step. Drop the folder at `/for/upago` on whichever domain Jake hosts from.
